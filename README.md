# NFL Data Visualization Dashboard & Analysis

## Data Sources

[NFL Big Data Bowl](https://www.kaggle.com/competitions/nfl-big-data-bowl-2025)

[Pro Football Reference](https://www.pro-football-reference.com/)

[NFL FastR Play-By-Play Data](https://github.com/nflverse/nflverse-pbp)

## Run it yourself

1. Clone Repository

```bash
git clone https://github.com/justin-hangoebl/nfl-data-viz.git
cd nfl-data-viz
```

2. Create Virtual Environment and install dependencies

```bash
uv venv .venv --prompt nfl
source .venv/bin/activate # or ".venv\Scripts\activate" on Windows
uv sync
```
